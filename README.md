## JSON FORMATTER
**A simple project for self-educational goal.**

**Project description:**

An app gets a JSON format as plain text and converts it by selected style ("Compact" - one line formatting, "Space-tab" - multiline formatting with tabulation). ***The app gets only valid json***.

**The project has a backend part:**
 - pure Java Core (jdk 15);
 - Servlets;

**And a very simple frontend:**
 - html, css
 - and a little bit of JSP :=)

**Servlet's container role** - Tomcat 9
